No input file specified. 

Esto ocurre por la configuración en el .htaccess, si has introducido modificaciones para poder sobreescribir las URLs sin incluir "index.php" en la URL y has usado la configuración por defecto que aparec en la guía de usuario de code igniter.

Para solucionar el problema debes usar el siguiente código en el .htaccess:

RewriteEngine on

RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteCond $1 !^(index\.php|images|robots\.txt)
RewriteRule ^(.*)$ index.php?/$1 [QSA,L]
