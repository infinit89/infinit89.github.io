Esta funcionalidad se denomina Instant Run.

Los requerimientos para que puedas usarlo son los siguientes:

  • Android Studio >=2.0
  • Android Plugin for Gadle >= 2.0.0
  • Dispositivo con Android >= 4.0 (API Level 15)
La primera vez que ejecutes tu aplicación con el icono de "Play" (o con Mayús + F10) todo se ejecutará de manera normal, en caso de permitirse el Instant Run aparecerá un pequeño rayo a la izquierda del icono (también en el de debug).
