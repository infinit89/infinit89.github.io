Flores tags: - programación - android - java modified_time: '2013-10-13T23:15:40.672+02:00' blogger_id: tag:blogger.com,1999:blog-8588071933588851396.post-3496976627157869505 blogger_orig_url: http://www.miguelg.com/2013/10/no-abrir-navegador-web-al-pulsar-enlace.html --- Si en una aplicación Android haces uso de alguna WebView y la URL que visitas hace alguna redirección o se pusa en algún enlace, ese enlace será capturado por el navegador y mostrado allí, de esta manera se muestra la URL en la barra del navegador. Si no quieres mostrar la barra de direcciones del navegador en una APP de android con WebView simplemente tienes que capturar las URLs para que las procese tu aplicación dentro de la WebView, para esto usamos la siguiente linea de código:

webView.setWebViewClient(new WebViewClient());  

Creando el método onCreate de la actividad que contenga el webView de esta manera:

    public void onCreate(Bundle savedInstanceState) {

        super.onCreate(savedInstanceState);
        setContentView(R.layout.webcontent);

        webView = (WebView) findViewById(R.id.webView);

        webView.setWebViewClient(new WebViewClient()); 
        webView.getSettings().setJavaScriptEnabled(true);

        webView.loadUrl("http://www.miguelg.com");

    }
