Flores tags: - seo - webmasters - google modified_time: '2013-03-20T07:35:41.998+01:00' blogger_id: tag:blogger.com,1999:blog-8588071933588851396.post-6300475075876303345 blogger_orig_url: http://www.miguelg.com/2013/03/que-piensa-google-de-las-webs-con-una.html --- Pregunta: ¿Qué piensa Google de las webs con una sola página? Se están convirtiendo más complejas y hay buenos sitios usando solamente una página (+ mucho CSS y JavasScript) trayendo la misma experiencia al usuario que las páginas tradicionales.

Respuesta: Actualmente hay muchas webs con buen contenido donde vas haciendo scroll indefinidamente y tienen todo el contenido y algunas cosas dinámicas molonas. Google ha conseguido mejorar manejando Javascript y hacer algunas cosas raras unusuales en interaciones de javascript, manejar cuando una parte de la página se pinta y cuando no, son buenos procesando eso. En general la idea sería ejecutar un test y no dejar todo el SEO en una página única que funcione bien si el CSS y el Javascript puede ser poco leido o bloqueado por robots.txt. Si está todo correcto no hay ningún problema con este tipo de páginas. Es un nuevo tipo de página, puede funcionar para ti según tu tipo de página, puede mejorar tu conversión o no. Si funciona para ti y para los usuarios tener todo en una sola página entonces funciona para Google también.
