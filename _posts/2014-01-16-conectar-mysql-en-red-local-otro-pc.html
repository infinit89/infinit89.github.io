--- layout: post title: Conectar MySQL en red local a otro PC date: '2014-01-16T19:30:00.001+01:00' author: Miguel González tags: - mysql modified_time: '2014-01-16T19:30:54.138+01:00' blogger_id: tag:blogger.com,1999:blog-8588071933588851396.post-303759819901025167 blogger_orig_url: http://www.miguelg.com/2014/01/conectar-mysql-en-red-local-otro-pc.html --- Si en una misma red local mantienes varios equipos, puedes hacerlos conectar dentro de la red entre ellos.

Para conectarte a otro PC con MySQL en red local tienes que dar permisos en ese servidor para que cualquier usuario (o la IP local de la máquina concreta pueda identificarse).

En el servidor debemos crear un usuario con todos los privilegios (puede limitarse, para este ejemplo daré permisos para todo) y que pueda conectarse desde cualquier ubicación (también puede limitarse por  IP o por su rango):

CREATE USER 'root'@'%' IDENTIFIED BY PASSWORD 'clave-identificacion';

GRANT ALL PRIVILEGES ON * . * TO 'root'@'%' IDENTIFIED BY PASSWORD 'clave-identificacion'

Cuando intentemos conectarnos a ese servidor a través del usuario "root" tendremos permisos.
